#  Customer Churn Prediction (End‑to‑End ML Pipeline)

Built a full churn‑prediction system for a subscription business, including modeling, calibration, and actionable decision rules.

---

##  Highlights

- Developed a **fully reproducible ML pipeline** (Python, scikit‑learn)
- Trained and compared **Logistic Regression, Random Forest, and Gradient Boosted Trees**
- Implemented **Platt scaling** for probability calibration
- Produced **OOF predictions**, **ablation table**, and **calibration curves**
- Designed a **manager‑ready retention rule** based on cost constraints
- Achieved **AUC 0.846 (OOF)** and **0.844 (holdout)**

---

##  Decision Rule

**Contact customers with `p_churn > 0.28`.**
